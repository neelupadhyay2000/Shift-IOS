SHIFT is a day-of timeline app for event professionals. Build your run-of-show, run it live, and when things run late the Ripple Engine recalculates everything in real time so you always know what’s next.

Getting startedBuild your first timeline

First, create your account: choose Continue with Email or Continue with Phone, type in the one-time code we send, and add your name to finish your profile (if you signed up with a phone number, we’ll also ask for an email). Then set a passcode to unlock the app and you’re ready to build.

1

Create an event

Tap + on the Events screen and add a title, date, and venue. Pick a venue and SHIFT automatically pulls in sunset, golden-hour, and weather for that day.

2

Add timeline blocks

Add blocks for each moment — ceremony, cocktails, golden-hour photos, dinner. Set a start time and duration, and add notes, a color, an icon, or a voice memo.

3

Tune each block

Mark must-not-move moments as pinned, set a minimum duration for blocks that can compress, and link dependencies so related blocks move together.

4

Go live

On the day, open the Live Dashboard. Complete blocks as they happen, and if you’re running behind, apply a shift — the engine recalculates the rest of the day instantly.

The core ideaHow shifting works

When one thing runs late, everything after it is affected. Instead of redoing the math by hand, you tell SHIFT how late you are and it ripples the change down the timeline — compressing flexible blocks to protect the moments that can’t move (like golden-hour photos). Try it:
